# AfriConnect Summit 2026
##  Projet
Site vitrine pour une conférence tech panafricaine fictive nommée **AfriConnect Summit**, un événement annuel qui réunit développeurs, entrepreneurs et investisseurs du continent.

Ce projet a été réalisé dans le cadre de l'examen de **Technologie Web** (HTML5 - CSS3 - JavaScript Vanilla - Git & GitHub).

## Auteur
- **Nom** : Diop Mouhamadou Moustapha
- **Classe** : L1 Reseaux Telecommunications
- **Année** : 2025-2026

## Technologies utilisées
- **HTML5** : Structure sémantique, accessibilité (alt, aria-label)
- **CSS3** : Flexbox, CSS Grid, Variables CSS, Animations, Transitions, Dark Mode, Responsive Design
- **JavaScript Vanilla** : DOM, Événements, IntersectionObserver, localStorage, Validation de formulaire
- **Git & GitHub** : Versioning, GitHub Pages

## Fonctionnalités JavaScript implémentées
1. **Dark Mode** avec localStorage (persistant entre les pages)
2. **Navbar dynamique** (scroll + hamburger mobile)
3. **Animations au scroll** (IntersectionObserver)
4. **Compteurs animés** (incrémentation au scroll)
5. **Compte à rebours** en temps réel
6. **Onglets du programme** (3 jours)
7. **Filtrage dynamique des intervenants**
8. **Validation de formulaire** (avec retour visuel)
9. **Bouton retour en haut** (apparaît après 300px)
10. **Année dynamique** dans le footer
